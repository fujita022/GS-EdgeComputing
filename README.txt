O projeto inicial foi um fundamento para que haja menores descartes de resíduos tóxicos e atóxicos no mar. Para que essa solução ocorra, foi desenvolvido um sistema
com sensor de temperatura (TMP36 Tinkercad), sendor de Ph(Foi usado um potenciômetro), um alarme(Utilizado para caso haja estado crítico com o mar (Temperatura for maior
ou igual a 33 graus celsius, e o pH menor ou igual a 5) Buzzer piezo). Para ser usado no Tinkercad é necessário alterar manualmente o TMP36 e o potenciômetro, onde respec-
tivamente alterará a temperatura e o pH. Conforme utilizado ele dará avisos que em tal local do oceano está com irregularidades podendo ser local de descarte ilegal de resí-
duos tóxicos.
